# Pós-Graduação em Tecnologia da Informação

Este repositório foi criado para organizar e documentar meus estudos
durante a pós-graduação em Tecnologia da Informação.

Aqui estão reunidos exercícios, resumos, exemplos práticos e mini-projetos
envolvendo desenvolvimento backend, banco de dados, web.

## Áreas de Estudo
- Java (POO, Collections, Exceptions...)
- Spring Boot (API REST, JPA, Hibernate)
- Banco de Dados (SQL, Modelagem, MySQL / SQL Server/ PostgresSQL...)
- Desenvolvimento Web (HTML, CSS, JavaScript)
- Git & GitHub
- Inglês Técnico para Tecnologia

## Objetivo
Construir uma base sólida para atuar como Desenvolvedor Backend
ou Analista de Sistemas / Suporte, unindo teoria e prática.

##  Organização do Repositório
Cada pasta representa uma disciplina ou área de estudo, contendo:
- Códigos-fonte
- Exercícios resolvidos
- Resumos em Markdown
- Exemplos práticos

## Em constante evolução
Este repositório é atualizado conforme o avanço das aulas e estudos.
